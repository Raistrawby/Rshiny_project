# Rshiny_project
Master Rshiny project.

## Requirements
- Clusterprofiler
- Pathview
- biomaRt
- shinyWidgets: `install.packages("shinyWidgets")`
- ggridges : `install.packages("ggridges")`
- ggupset: `install.packages("ggupset")`
- plotly `install.packages("plotly")`
- tidyverse `install.packages("tidyverse")`
